# Začínáme

SkillBrain je MCP server, který přidává trvalou paměť vašemu AI coding toolu. Po nastavení si Claude, Cursor nebo Cline pamatují konvence projektu mezi sezeními.

## Co je SkillBrain

Pokaždé, když spustíte sezení s AI coding toolem, kontext začíná od nuly. SkillBrain to řeší ukládáním architektonických rozhodnutí, projektových vzorů, oprav chyb a dovedností týmu do grafu. AI je načítá automaticky při spuštění.

## Požadavky

- Node.js 18 nebo vyšší
- pnpm, npm nebo yarn
- AI coding tool kompatibilní s MCP: Claude Code, Cursor, Cline nebo Windsurf

## Rychlý start

```bash
npx skillbrain init
```

Pro ověření funkčnosti:

```bash
npx skillbrain status
```

Očekávaný výstup: `SkillBrain running on port 3001`.
