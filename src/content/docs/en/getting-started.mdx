# Getting Started

SkillBrain is an MCP server that adds persistent memory to your AI coding tool. Once configured, Claude, Cursor or Cline remember your project conventions across sessions.

## What is SkillBrain

Every time you start a session with your AI coding tool, context starts from scratch. SkillBrain solves this by storing architectural decisions, project patterns, bug fixes and team skills in a graph. The AI loads them automatically on startup.

## Prerequisites

- Node.js 18 or higher
- pnpm, npm or yarn
- An MCP-compatible AI coding tool: Claude Code, Cursor, Cline or Windsurf

## Quick start

```bash
npx skillbrain init
```

To verify it's running:

```bash
npx skillbrain status
```

Expected output: `SkillBrain running on port 3001`.
